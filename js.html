Curry Function Example
function curry(a){
 return function(b){
  return function(c){
   console.log(a+b+c);
   return a+b+c;
  }
 }
}
let out = curry(5)(6)(4);
output => 15

Lexical Scope vs Closure
  • The lexical scope allow a function scope to access statically the variable from the outer scopes
let globalVar = 20;
const outerFun = (num1) => {
 let num2 = 10;
 const innerFun = () => {
  let sum = num1+num2+globalVar;
  console.log(sum);
 }
 innerFun();
}
outerFun(5);
output => 35

  • Closure defination 1 => a Closure is a function that captures variable from it's lexical scope.
  • Closure defination 2 => inner function can have access to the outer function variable as well as all the global variable
let globalVar = 10;
const outerFun = (num1) => {
 let num2 = 15;
 const innerFun = () => {
  console.log(num1+num2+globalVar);
 }
 return innerFun
}
let inner = outerFun(10);
// console.dir(inner); it will give you innerFun detail.
inner();
output => 35

Type of Error
  • Load time errors: Errors that come up when loading a web page, like improper syntax errors, are known as Load time errors and generate the errors dynamically.
  • Runtime errors: Errors that come due to misuse of the command inside the HTML language.
  • Logical Errors: These are the errors that occur due to the bad logic performed on a function with a different operation.

A function that is declared without any named identifier is known as an anonymous function. In general, an anonymous function is inaccessible after its declaration.

shallow copy of object Ref

A deep copying means that value of the new variable is disconnected from the original variable while a shallow copy means that some values are still connected to the original variable.

let obj = {
 name:"prashant",
 address:{
  city:"gwalior"
 }
}

let objCopy = Object.assign({}, obj);
objCopy.name = "mini";
objCopy.address.city = "seondha";

console.log('Original Object ===>', obj); console.log('Copy Object ===>',objCopy);

Original Object ===>
{
 name: 'prashant', // parent key value will be seprate or both object
 address: {
  city: 'seondha' // child key value will be same
 }
}

Copy Object ===>
{
 name: 'mini', // parent key value will be different
 address: {
  city: 'seondha' // child key value will be same
 }
}

preventDefault

The preventDefault() method cancels the event if it is cancelable, meaning that the default action that belongs to the event will not occur.

document.getElementById("myAnchor").addEventListener("click", function(event){
 alert('before');
 event.preventDefault();
 alert('after');
});

alert('before');
