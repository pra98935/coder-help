Angular Universal use
  • SEO purpose, if you will check source code for any page without integrate universal it will show you main index.html file and during google crawl content will not read that's why need server side rendering. after integrate universal it will give source code for exact page
  • show the first page quickly
  • Improve performance in mobile
Basic Setup
  • ng add @nguniversal/express-engine after run this command, some server related files will be create
  • npm run build:ssr after run this command dist folder will create. it will contain some folder like browser & server
  • npm run serve:ssr it will provide you new port for your app
Note
  • if you want to change port for SSR application, path => project folfer => server.ts
